#  PC Troubleshooting Wizard

An interactive browser-based troubleshooting assistant designed to simulate first-level IT Support diagnostic workflows.

Users can select common technical issues and receive structured troubleshooting recommendations.

## Features

* Interactive Issue Selection
* Guided Troubleshooting
* Dark Mode
* Simple User Interface

## Scenarios

###  No Internet

* Restart adapter
* Reconnect network
* Run troubleshooter

###  Printer Offline

* Restart printer
* Set default printer
* Restart spooler

###  PC Running Slow

* Disable startup applications
* Run Disk Cleanup
* Restart device

###  Cannot Login

* Verify account
* Reset password
* Test login

## Technologies

* HTML5
* CSS3
* JavaScript

## Objectives

* Strengthen IT troubleshooting logic
* Improve structured issue resolution
* Simulate IT Support diagnostics
